O aplikaci

PS Fitness Gym System je moderní webová aplikace určená pro správu posiloven, která umožňuje jednoduchou evidenci členů, sledování jejich docházky a generování týdenních reportů. Systém je vytvořen pomocí frameworku Django v rámci závěrečného projektu kurzu Python Web Development.

Hlavní funkce

  • Registrace a přihlášení uživatelů
  • Správa členství s filtrováním a vyhledáváním
  • Online docházkový systém (AM/PM)
  • Automatický výpočet expirace časových členství
  • Přehledný týdenní report s vizualizací

Technologický stack

  • Backend: Python, Django, SQLite
  • Frontend: HTML5, CSS3, Bootstrap
  • Styling: Vlastní CSS + responzivní design
  • Autentizace: Django Auth System

Cílová skupina

Menší a střední posilovny, které potřebují jednoduché a přehledné řešení pro správu členů a docházky bez nutnosti placených služeb.

Autor projektu

David Hruška — student kurzu Python Web Development.
Projekt vznikl jako závěrečné zadání v roce 2025.
